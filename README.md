# Unturned PHP Status

# Install
- Download
- Extract
- Configure config.php (They're 3 samples inside the config, you can simply copy and paste for any additional servers.)
- Upload to webserver

# Requirements
PHP 5.5+ 64-bit
Web server must allow UDP connections
